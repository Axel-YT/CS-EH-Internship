üß© TASK 1: FILE INTEGRITY CHECKER
Description:

The File Integrity Checker is a security tool designed to detect unauthorized changes in files. The main objective of this task was to understand how files can be monitored using cryptographic hash functions.

In this project, I created a Python-based program that calculates the hash value (SHA-256) of files and stores them as a baseline. During later execution, the program recalculates the hash values and compares them with the original values. If any file is modified, deleted, or newly added, the program detects and reports it.

This task helped me understand:

How hashing algorithms work

Importance of data integrity

File system monitoring techniques

This type of system is commonly used in antivirus software, intrusion detection systems, and data integrity tools.
